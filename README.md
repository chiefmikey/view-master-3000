# View-Master 3000

_Reddit client for scrolling media content by user or subreddit_

### User

```url
viewmaster3000.com/u/{username}/{media?}
```

### Subreddit

```url
viewmaster3000.com/r/{subreddit}/{filter?}/{media?}
```

### Top

```url
viewmaster3000.com/r/{subreddit}/top/{time?}/{media?}
```
